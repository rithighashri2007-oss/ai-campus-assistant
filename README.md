# AI Campus Assistant 🎓🤖

This project is developed as part of a Tech Club selection task.

## Objective
To build an intelligent AI assistant that helps students navigate campus life by answering college-related queries.

## Planned Levels
- Level 1: Basic AI Assistant (LLM-based Q&A)
- Level 2: Knowledge-Aware Assistant (RAG with college documents)
- Level 3: Agentic Assistant (Autonomous behavior – future scope)

## Tech Stack
- Python
- AI API (to be added)
- Vector Database (for Level 2)
